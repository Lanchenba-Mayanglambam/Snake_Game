# Snake Game 🐍

A classic Snake Game built using HTML, CSS, and JavaScript.

## 🎮 Features
- Grid-based movement
- Score & High Score (localStorage)
- Sound effects
- Increasing speed

## 🛠️ Tech Stack
- HTML5 Canvas
- JavaScript
- CSS

## 🚀 How to Run
Open index.html in your browser
